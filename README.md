# 戦国AIエージェントTCG（仮称『軍師』）

**令和から戦国に転生した軍師となり、Claude Code のコマンドを打って天下を狙うカードゲーム。**
表は「ポケポケ」と同じ手触りのカードバトル、裏はAIエージェントに命令を出して武器や有利を作る——その二刀流が肝です。

> これは **テストプレイできるMVP（最小で遊べる一式）** です。バランスや絵は遊びながら調整していきます。

---

## ▶ 遊び方（起動）

**いちばん簡単：`index.html` をダブルクリック**してブラウザで開くだけ。インストール不要です。

> うまく動かない場合（ブラウザのセキュリティで止まる等）は、簡易サーバーで開いてください：
> ```powershell
> cd C:\dev\sengoku-agent-tcg
> python -m http.server 8000
> # → ブラウザで http://localhost:8000 を開く
> ```

最初に「遊び方」ボタンで3分チュートリアルが読めます。

---

## 🎮 3つのモード

最初の画面で選べます。

- **対CPU**：AI軍師と一戦。
- **1人二役（ホットシート）**：1台で両陣営を交互に操作。**両者の挙動を全部自分で確かめたいとき向き**（サーバー不要）。
- **観戦（AI対AI）**：AI同士の対戦を眺める。右の戦記とワークツリーで両者の手が追える。

## 🎴 ゲームの流れ（ざっくり）

1. **モードと家を選ぶ**（織田家＝速攻と鉄砲／武田家＝騎馬突撃）。
2. **開幕の布陣**：軍配（先攻後攻）→手札5枚→**先鋒に1体（必須）・後備えに最大3体を自分で選ぶ**。
3. **無名の兵士を場に出す**（足軽・騎馬武者など＝たね）。カードは**クリックで詳細を読んでから**使う。
3. **兵を配る**→ 武将のワザを撃って戦う。攻撃すると自分の番が終わる。
4. **出世（進化）**：無名の兵士に「出世カード」を重ねると、明智光秀→織田信長…と**名を挙げて強くなる**。
5. **⚑ 軍師の命令**（裏）：`/plan` `/code-review` など本物のコマンドを発注。脳容量（コンテキスト）を使い、数手番で完成して戦況に効く。
6. **首級3つ**取れば勝ち（雑兵1・侍大将2・大名3）。

右のサイドパネルで、**将軍→家臣→陪臣のワークツリー**（AIエージェントの並列作業）が見えます。

## 🗂 ファイル構成

```
sengoku-agent-tcg/
├ index.html              ← これを開けば遊べる
├ styles/
│  ├ tokens.css           デザイントークン（和紙・墨・朱・明朝）
│  └ game.css             盤面・カード・パネルの見た目
├ src/
│  ├ data/                カード・命令札・デッキの中身（ここを足せばカードが増える）
│  ├ engine/engine.js     ルール・戦闘・出世・状態異常・裏の命令層
│  └ ui/ui.js             画面の描画と操作
├ art/
│  ├ prompts/             全キャラのGPT Image 2プロンプト（.md と .json）
│  ├ generate_images.py   画像を一括生成するスクリプト
│  ├ Codexで一括生成する手順.md
│  └ images/              ← 生成した武将の絵を置く場所（無くても遊べる）
├ docs/                   要件定義・命令札図鑑
└ test/                   エンジン検証用（node test/sim.js）
```

---

## 🖼 武将の絵を入れる

最初は絵なし（家紋色の枠＋名前）で遊べます。浮世絵タッチの絵を入れるには：

1. `art/Codexで一括生成する手順.md` を開く
2. APIキーを通して、コピペ用プロンプトを Codex に貼る（または `python art/generate_images.py`）
3. `art/images/` に絵が入ると、ゲームを開き直すだけで**自動でカードに表示**されます

## 0. 確定した世界観・立場（v2で追加）

- **世界観**：プレイヤーは **令和から戦国に転生した軍師**。Claude Code を使える知識を持って戦国の世に現れ、コマンドを打って戦略家として戦う。
- **プレイヤーの立場＝二役**：
  - **将軍**として、武将カードを動かし殴り合う（表のカードバトル）。
  - **軍師**として、AIエージェントに命令を出し武器・有利を作る（裏の命令層）。
  - 表のカードバトルに、軍師の役割が**パラレルに合わさる**のが本作の構造。

## 1. 表＝合戦盤（ポケポケ準拠）

- 盤面：**先鋒1・後備え3・兵站庫・陣容（山札・右）・討死（トラッシュ）・手札・陣形（場に1枚）**。
- 資源「**兵**」：毎ターン兵站庫に自動で1たまり、武将へ配備（1ターン1回）。ワザのコスト・退き口に使う。
- カード種別：**武将／装備（1人1点）／采配（軍需品=何枚でも・軍師の采配=1ターン1枚）／陣形（場に1枚）**。
- 勝敗：**首級3点先取**（雑兵1・侍大将2・大名3）。または相手の場が全滅。
- 相性（弱点）：+20の固定値。抵抗なし（計算がやさしい）。
- 状態異常：毒矢・火攻め・油断・金縛り・混乱（退き口・出世で解除）。

## 2. 出世（進化）— v2で確定した中核設計

- **無名の兵士（たね）→ 名を挙げて有名武将（侍大将）→ 大名（伝説）** の3段階。
- たねは**人格のない兵種**（足軽・騎馬武者・槍足軽・弓兵・鉄砲足軽）。デッキごとに、ここから有名武将へ出世する。
  - 例：足軽 →（出世）→ 明智光秀 →（出世）→ 織田信長。
  - 例：足軽 → 木下藤吉郎 → 豊臣秀吉（裸一貫からの出世そのもの）。
- 「**生産性（エージェントをうまく回す）を高めると出世する**」という思想を、出世で部下（後備え）が増え強くなる流れで表現。
- 1ターン1段階・出した直後のターンは出世不可（ポケポケ準拠）。出世で状態異常リセット・兵力上限UP・損耗は引き継ぎ。

## 3. 裏＝軍師の命令層（オリジナル）

- 資源「**コンテキスト予算（脳容量・既定100）**」：命令を発注すると消費し、完成で戻る。
- **命令札＝本物の Claude Code / Codex コマンド**（名はアルファベットのまま）。MVPは10種：
  `/plan`（軍議）`/code-review`（軍師の添削）`/security-review`（忍び探索）`codex exec`（奇襲）`/compact`（兵站圧縮）`--sandbox`（結界）`/agents`（軍団編成）`/rewind`（時を遡る）`/model`（軍師交代）`/memory`（家訓）。
- 命令は**数手番かけて完成**し、完成すると効果が盤面に投入される（攻撃・バフ・諜報・回復・予算回復・並列増・防御など）。
- **サブエージェント（家臣）＝worktree（別働隊の陣地）** で並列作業。**将軍→家臣→陪臣**の指揮系統がサイドパネルに**ワークツリー**として見える。
- 進捗は**各ターン終了ごとに1段階**（`/model` 完成後は2段階）進む。

## 4. 画面（実装済み）

- 横画面ワイド。左＝合戦盤（上：敵／下：自分、中央で先鋒が対面、後備え横一列、山札は右、討死は左、手札は最下部）。右＝**サイドパネル（軍師の陣）**＝コンテキスト予算・発注中の命令・ワークツリー・戦況メタ・戦記。
- スマホ幅ではパネルが下に回る。

## 5. デザイン（実装済み）

- `DESIGN.md`（明朝・和紙白・墨罫・単色アクセント・影/角丸なし・文字は箱に収める）を**UIの規律**として継承。
- アクセントは**朱**、伝説は**金**、勢力は藍・群青・金茶で識別。
- **全テキストは箱（カード・帯）に収め、はみ出し・意図しない折返しを禁止**（`nowrap`/`clip`/行数clampで実装）。
- カードの絵だけ**フル浮世絵（GPT Image 2）**。枠（規律）×絵（絢爛）の対比。

## 6. チュートリアル・分かりやすさ（v2で追加）

- 普通の人には難しいゲームなので、**3分チュートリアル**を実装（6枚の要点カード）。
- 操作はクリック方式で、選んだ手札→対象クリックで確定。誤操作はトーストで理由を表示。
- 効果は戦記（ログ）と数値で明示。今後さらにアニメ演出を足す。

## 0. 世界観・立場

- プレイヤーは **令和から戦国に転生した軍師**。Claude Code を使える知識で戦国を戦う。
- **二役**：**将軍**として武将を動かし殴り合う（表）＋**軍師**としてAIエージェントに命令を出し有利を作る（裏）。表と裏が1画面で並走するのが本作の核。

## 1. 表＝合戦盤（ポケポケ準拠＋戦国アレンジ）

- 盤面：**先鋒1・後備え3・本陣（大名）・兵糧の在庫・山札・捨札・手札・陣形（場に1枚）**。
- カード種別：**武将／装備（1人1点）／采配（軍需品=何枚でも・軍師の采配=1ターン1枚）／陣形（場に1枚）**。
- 相性（弱点）：+20固定。状態異常：毒矢・火攻め・油断・金縛り・混乱（退き口・出世で解除）。

### 勝敗（v3の中核）
- **相手の武将を3体討ち取れば勝ち／自分が3体討たれたら負け**（`warlordsLost >= 3`）。

### 大名＝本陣の切り札（v3の中核）
- 大名（信長/信玄/謙信/家康/秀吉）は**デッキに入れず本陣に控える**。
- **味方が2体討たれた時点で、3体目の砦として自動出陣**。**本陣にいる間は無傷**＝満タンで登場し、終盤の見せ場になる。大名を討てば3体目＝決着。

### 兵糧（行動資源）
- 毎ターン在庫+2。武将へ送って蓄え、ワザで消費。**同じ武将へは1ターンに1つまで**（移動して二重に付ける穴を封鎖）。
- 退き口（交代）は兵糧消費（**種1・侍大将2・大名1**／采配旗を装備で無料）。

## 2. 出世（進化）

- **無名のたね（足軽・騎馬武者・槍足軽・弓兵・鉄砲足軽）→ 名を挙げた侍大将（明智など）** の1段階のみ。名持ち→名持ちの進化は無い。
- デッキは「**進化する種＋進化しない重い単体**」で構成（種が腐らないよう侍大将は厚め×3）。
- 1ターン1段階・出した直後の手番は出世不可。出世で状態異常リセット・兵力上限UP・損耗（ダメージ）は引き継ぎ。

## 3. 裏＝軍師の命令層（オリジナル）

- 資源「**脳容量（コンテキスト・既定100）**」。自然回復は控えめ（+5/turn）。**完遂しても戻らない**＝「ここぞ」で撃つ資源。
- **命令札11種**（名はアルファベット＝本物のClaude Code/Codexコマンド）：
  `/plan`（軍議）`/code-review`（軍師の添削）`/security-review`（忍び探索）`codex exec`（奇襲の一撃）`/compact`（兵站圧縮）`--sandbox`（結界）`/agents`（軍団編成）`/rewind`（時を遡る）`/model`（軍師交代）`/memory`（家訓）`/deep-research`（総力調査・4手番の強力技）。
- メニューで**複数まとめて発令**でき、**自分の手番が終わるたびに**進行して数手番で完成→効果が盤面に投入。
- **サブエージェント（家臣）＝worktree（別働隊の陣地）** で並列作業。**軍師→家臣→陪臣**の指揮系統が右パネルに**ワークツリー**として見える。

## 5. モード・UI・演出

- モード：**対CPU**（プレイヤーだけ軍師・CPUは兵力+20）／**1人二役（ホットシート）**／**観戦（AI対AI）**。
- 画面：左＝操作バー、中央＝合戦盤（相手先鋒は小型・自分は大きくワザ付き・ベンチは枠で囲い常時可視）、右＝サイドパネル（軍師の陣）。**盤は必ず画面内**・操作でガタつかない。
- 演出：Web Audio和風効果音（太鼓/斬撃/琴/法螺貝/銅鑼）・攻撃の突き・被弾の揺れ・ダメージ数字・討死バナー・配札フリップ・レアの金光沢・音ON/OFF。
- 分かりやすさ：3分チュートリアル・デッキガイド・カード詳細（読んでから使う）・誤操作はトーストで理由表示。

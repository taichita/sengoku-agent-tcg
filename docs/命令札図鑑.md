# 戦国AIエージェントTCG ── 命令札図鑑（付録・調査成果）

> ⚠ **これは「命令札のネタ帳（在庫）」です。実装の正は `src/data/commands.js` とゲーム内の命令メニュー。** 数値・採用枚数はこの図鑑より実装が新しいことがあります。新しい命令札を足すときの発想源として使ってください（実装済みは11種）。

> 本物の Claude Code / Codex のコマンドを **441項目** 調べ上げ、ゲームの「命令札（軍師がAIエージェントに出す命令）」として使える形に整理した一覧です。要件定義書の第5章・第14章の付録にあたります。

> - **コマンド名はアルファベットのまま**（訳さない）。「ゲーム案」は戦国カードとしての読み替え案。
> - 2026年6月時点で事実検証済み。Claude Code 公式コマンドは約100種、ここではゲームに使える代表を厚めに収録。

> - 実装時はこの中から MVP 用に 8〜15 枚を選んで先に作ります。
